# Data-Engineering-Case-Study-G2
Case Study
You've been selected to complete the case study for the position of data engineer at Deloitte. This case study will be used to evaluate your level of experience, as well as technical and soft skills.
Background
You've been tasked with collecting data and coming up with insights regarding Airbnb listings in Cape Town and identifying if there is any correlation between ratings and the weather. It may be beneficial if you can identify any additional insights and present them. If there are any sections you find difficult to complete, you are encouraged to submit whatever results you have been able to achieve so far, or you may provide alternative submissions that showcase related efforts.
